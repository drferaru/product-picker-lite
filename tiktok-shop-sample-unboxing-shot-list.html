Do not skip the boring proof shots

  • Package condition before opening.
  • Scale beside a hand, phone, table, or doorway.
  • All included parts, instructions, and variants.
  • First-use friction, defects, noise, smell, or mess.

Sample unboxing verdict:
- Package arrived clean? yes / no
- Listing matched the actual item? yes / no
- Size is obvious on camera? yes / no
- First use was simple? yes / no
- Before/after proof is visible in 3 seconds? yes / no
- Any defect or return-risk closeups? yes / no
- One vertical hook is usable? yes / no

Decision:
0-1 weak answers = run Product Picker Lite with the footage.
2 weak answers = ask supplier follow-up or order one replacement sample.
3+ weak answers = reject before inventory or creator spend.

The sample unboxing is the bridge between sourcing and content proof. If the footage is strong, use it to score the product. If the product passes, turn the raw proof into three angles before paying for inventory or creators.
